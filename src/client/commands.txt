<========Дополнения=======Olegmmm=======>
В игре используются следующие контролы:

Esc - выход
1,2,3 - смена дня и ночи. потом уберу, менять будет сервер одновременно всем.
h - включить/выключить показ гексов
Ctrl+клавиши курсора влево-вправо - вертеть чузена

Enter - режим командной строки:

в этом режиме доступны следующие команды

конец - выход из игры, сокращать нельзя
все остальные команды можно сокращать хоть до одной буквы пример: "г привет"
говорить - сказать что-то
кричать - кричать и орать будут слышны дальше, и визуально выделяются заглавными буквами
орать
эмоция - выразить эмоция например строка "эм глубо задумался" приведет к появлению 
**Олег глубоко задумался**
 - если вас Олег зовут конечно :)

кроме команд можно использовать социалы (socials.txt). есть 3 вида исползования
на примере социала "улыбнуться"
улыбнуться
Вы тепло улыбнулись

улыбнуться я
Вы улыбнулись своим мыслям

улыбнуться кош
Вы тепло улыбнулись Кошу

все 3 вида использования для каждого (почти) социала будут давать разный эффект. Причем вы увидите одно предложение, цель социала другое, а все вокруг - третье.
<========Дополнения=======Den Baster=======>
Клавишы
O - отладочная инфа
ASDWQE - клавиши ходьбы.
К - достать Нож(Нужна дифференсация по оружию, потом) (Только для клиента)
J - убрать Нож (Только для клиента)
P - атака (Только для клиента)

space(не сделано) - отмена длинного действия(сейчас действия очень короткие)
M - Мировая карта.
Для машин 2 Гб,248 Озу с встроенным видео существенный спад при отображении мировой карты, падение до 2 FPS 1024x768. Без интерфейса 1024x768 30 FPS, 800х600 60 fps.

ОЧЕНЬ ВАЖНО! надо настраивать fonline.cfg для каждого компа индивидуально чтобы достичь лучшей производительности.
